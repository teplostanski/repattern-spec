# repattern Specification

Specification of an embedded DSL for building JavaScript RegExp from declarative schemas.

## Abstract

The repattern specification defines an [embedded DSL](https://en.wikipedia.org/wiki/Domain-specific_language) for building JavaScript regular expressions from declarative [`Schema`](./src/content/spec/current/en.md#21-schema) objects that are converted into `RegExp` instances.

Schemas provide a human-readable way to describe patterns, as an alternative to writing regular expressions manually.
